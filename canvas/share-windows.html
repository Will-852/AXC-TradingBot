Windows Setup Guide

Windows 版本支持 Dashboard 睇數據同 AI 分析。
自動排程（LaunchAgents）同自動交易係 macOS 專屬功能。
Download Setup Package (Today's Build)

Contains all required files — no API Keys or personal data included

GitHub Latest Release (Recommended)
Prerequisite: Install Python 3.11+ first. Make sure to check "Add python.exe to PATH" during installation.

Installation Steps (approx. 10 min)

1
Open PowerShell
Press Win+X then select Terminal (or search "PowerShell" in Start Menu)
2
Extract Files
Download the ZIP, then paste this in PowerShell:
$dest = "$HOME\.openclaw" New-Item -ItemType Directory -Force -Path $dest Expand-Archive -Path "$HOME\Downloads\openclaw-setup-*.zip" -DestinationPath $dest -Force
3
Create Config File
Copy-Item "$HOME\.openclaw\secrets\.env.example" "$HOME\.openclaw\secrets\.env"
4
Fill in API Key
Open the config file in Notepad and fill in your PROXY_API_KEY:
notepad "$HOME\.openclaw\secrets\.env"
Fill in the key value, then Ctrl+S to save, close Notepad.
5
Install Dependencies
pip install -r "$HOME\.openclaw\requirements.txt"
Wait 1-2 minutes. You should see Successfully installed when done.
6
Start Dashboard
cd "$HOME\.openclaw" python scripts\dashboard.py
Then open your browser: http://localhost:5555
(Keep the PowerShell window open while Dashboard is running)
✓
Verify Installation
Open http://localhost:5555 in your browser. If the dashboard loads, you're all set!
Note: health_check.sh is a macOS script and won't run on Windows. Just check the Dashboard loads correctly in your browser.

Windows Notes

What works on Windows?
Dashboard (full UI), AI analysis, market data display, agent status monitoring. Everything you see in the browser works the same as macOS.
What doesn't work on Windows?
LaunchAgents (macOS auto-scheduler), Scanner auto-run, and Trader auto-execution. These require macOS. On Windows, you run the Dashboard manually from PowerShell.
Python version?
Python 3.11 or newer. Download from python.org. During install, check "Add python.exe to PATH".
PowerShell says "execution policy" error?
Run this once as Administrator:
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
"pip" not recognized?
Python wasn't added to PATH. Reinstall Python and check the PATH checkbox, or use py -m pip install ... instead.
Something went wrong?
Send a screenshot of the error to the friend who shared OpenClaw with you.
