系統架構

Scanner ───→ Analyst ───→ Decision ───→ Trader ↑ ↓ Heartbeat Telegram Notification News Agent ───→ Sentiment ───→ Context Layer
  • 10 Agents (main / scanner / trader / heartbeat / binance_scanner / binance_trader / news_agent / ...)
  • 18 Scripts — Python 執行層（零 token 消耗）
  • 10 LaunchAgents — macOS 自動排程
  • 推理層：Claude API（Haiku / Sonnet）
