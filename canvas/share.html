安裝步驟（大約 10 分鐘）

1
解壓文件
下載完成後，打開 Terminal（Mac 搜尋「Terminal」），貼上：
mkdir -p ~/axc-trading unzip ~/Downloads/axc-setup-*.zip -d ~/axc-trading/
2
複製設定文件
cp ~/axc-trading/secrets/.env.example ~/axc-trading/secrets/.env
3
填入 API Key
用以下指令打開文件，填入你的 PROXY_API_KEY：
nano ~/axc-trading/secrets/.env
填完按 Ctrl+X → Y → Enter 儲存退出
4
安裝依賴
pip3 install -r ~/axc-trading/requirements.txt
等待約 1-2 分鐘，看到 Successfully installed 就完成
5
啟動 Dashboard
cd ~/axc-trading && python3 scripts/dashboard.py &
然後打開瀏覽器： http://localhost:5555
✓
確認安裝成功
bash ~/axc-trading/scripts/health_check.sh
看到 39 pass / 0 fail 代表一切正常

需要哪些 API Key？

Key 名稱用途去哪申請
PROXY_API_KEY AI 大腦（決策分析） 問介紹你的朋友 / console.anthropic.com 必填
PROXY_BASE_URL AI Proxy 地址 填 https://tao.plus7.plus/v1 必填
ASTER_API_KEY/SECRET 交易所下單 Aster Exchange → API 管理 交易必填
TELEGRAM_BOT_TOKEN 手機通知+遙控 Telegram 搜尋 @BotFather 選填
BINANCE_API_KEY/SECRET Binance 行情 Binance → API 管理 選填
VOYAGE_API_KEY AI 記憶搜尋 voyageai.com（免費） 選填
API Key 只填到 secrets/.env，唔好分享給任何人，唔好放上網。

常見問題

安全嗎？你會唔會攞到我的 Key？
Setup Package 唔包含任何 API Key。你的 Key 只存在自己電腦的 secrets/.env 文件，唔會上傳任何地方。
費用係點？
AXC 本身免費開源。需要付費的只有 Claude API（按用量，Proxy 通常更便宜） 同交易所手續費。
Windows 得唔得？
而家主要支持 macOS。Windows 可以跑 Dashboard 睇數據， 但自動排程（LaunchAgents）係 macOS 專屬。
出現錯誤點算？
先跑健康檢查：
bash ~/axc-trading/scripts/health_check.sh
再睇完整安裝指南：http://localhost:5555/details
