點樣匯入舊結果？
1. 撳上面「匯入」按鈕，選 .json 文件
2. 或者手動放 JSONL 文件到 backtest/data/，命名為 bt_BTCUSDT_60d_trades.jsonl
3. 「載入舊結果」就會自動列出
匯入 JSON 格式
{ "config": { "symbol": "BTCUSDT", "days": 60, "balance": 10000 }, "trades": [ { "side": "LONG", "entry": 84500, "exit": 85200, "entry_time": "2026-01-15T08:00:00", "exit_time": "2026-01-15T14:00:00", "pnl": 700, "exit_reason": "TP" } ] }
必填：config.symbol + 每筆 trade 嘅 side, entry, exit。pnl 可省略（自動計算）。
