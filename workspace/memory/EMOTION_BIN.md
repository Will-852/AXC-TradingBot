## 使用指引

每次記錄格式：
```
### [YYYY-MM-DD] [類型] — [一句總結]
- 發生咗咩：[事實]
- 點解會咁：[根本原因]
- 情緒反應：[當時嘅「感覺」— 急、怕、貪、猶豫...]
- 教訓：[下次點做]
- 狀態：[已消化 / 仍在處理]
```

類型標籤：
- `LOSS` — 倉位虧損
- `MISSED` — 錯失入場
- `PANIC` — 恐慌操作
- `GREED` — 貪心加倉/唔走
- `HESITATE` — 猶豫錯過
- `SYSTEM` — 系統/技術錯誤
- `OVERRIDE` — 人手覆蓋策略（通常係錯）
